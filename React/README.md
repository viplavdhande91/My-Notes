
# React

React Commands


## Commands and Steps

#### 1. Create new app

```http
npx create-react-app my-app
```


#### 2.TO RUN THE APP:  In index.js, open the terminal and write
```http
 npm start
 ```

#### 3.TO CREATE APP FOR AND PRODUCTION:
```http
npm run build
 ```
